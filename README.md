### Installation

1. Clone the repository:

   ```sh
   git clone [repository url]
   cd repository-name
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start Expo DevTools:
   ```sh
   npm start
   ```
   or
   ```sh
   expo start
   ```
